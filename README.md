# LnG Motors | Premium Pre-Owned Luxury Cars

A sleek, responsive, and fully interactive front-end web application built for a premium pre-owned luxury car dealership based in Nairobi, Kenya. This project delivers an elite digital showroom experience, structured neatly using modern semantic layout techniques and interactive components.

---

## 🚀 Key Features

* **Responsive Showroom Banner:** A high-end luxury hero section configured with custom fluid CSS scaling (`background-size: cover`) ensuring structural stability across all viewports.
* **Interactive Search & Filter Grid:** Dynamic filtering layout component to easily sort through featured vehicle listings (Mercedes-Benz, BMW, Audi).
* **Live Currency Valuation Converter:** A custom programmatic JavaScript utility that translates global portfolio baseline pricing ($ USD) into local currency (KES Sh) using live calculation matrices.
* **Complete Corporate Profiles:** Full-featured Multi-Page architecture (`index.html`, `about.html`, `inventory.html`, etc.) integrating optimized clean pathing for team directories and architectural assets.

---

## 🛠️ Technology Stack

* **Structure:** HTML5 (Semantic Elements)
* **Styling & Layout:** Bootstrap v5.3.3 & Custom CSS3
* **Icons:** FontAwesome v6.4.0
* **Logic Modules:** Vanilla JavaScript (ES6+ Document Object Model API)
